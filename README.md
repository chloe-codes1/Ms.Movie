## Install

<br>

### Backend

```bash
$ cd backend
$ python install -r requirements.txt
$ python manage.py migrate
```

<br>

### Frontend

```sh
$ cd frontend
$ npm install
```

<br>

<br>

## Usage

<br>

### Backend

```bash
$ cd backend
$ python manage.py runserver
```

<br>

### Frontend

```sh
$ cd frontend
$ npm run serve
```

<br>

## Run tests

```sh
$ cd frontend
$ npm run test
```
